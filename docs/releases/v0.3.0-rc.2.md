# Amiweave v0.3.0-rc.2

This release candidate introduces the Amiweave public brand and connects packaged desktop clients to `https://amiweave.com`.

## Downloads

- Windows x64: use the `.exe` installer.
- macOS Intel: use the `mac-x64.dmg` image.
- macOS Apple Silicon: use the `mac-arm64.dmg` image.
- Verify downloads against `SHA256SUMS.txt`.

## Code Signing

Windows releases accepted into the open-source signing program will use **free
code signing provided by SignPath.io, certificate by SignPath Foundation**. See
the [Amiweave Code Signing Policy](https://github.com/erick-huazi/LianYuPC/blob/main/CODE_SIGNING_POLICY.md).

The `v0.3.0-rc.2` candidate was published before the SignPath application and
remains unsigned. It is not represented as a SignPath-signed artifact.

## Highlights

- Isolated Singapore production deployment behind Cloudflare Full (strict) TLS.
- OpenAI-compatible providers, SillyTavern character cards, memory privacy controls, and Lite mode.
- Windows and macOS desktop packages with ASAR integrity verification.

## Candidate Build Notice

These artifacts are not yet signed with trusted Windows or Apple developer certificates. Windows Smart App Control may block the Windows installer, and macOS Gatekeeper may block the macOS package. Trusted production signing and Apple notarization remain required before the stable release.
