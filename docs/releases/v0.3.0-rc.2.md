# Amiweave v0.3.0-rc.2

This release candidate introduces the Amiweave public brand and connects packaged desktop clients to `https://amiweave.com`.

## Downloads

- Windows x64: use the `.exe` installer.
- macOS Intel: use the `mac-x64.dmg` image.
- macOS Apple Silicon: use the `mac-arm64.dmg` image.
- Verify downloads against `SHA256SUMS.txt`.

## Highlights

- Isolated Singapore production deployment behind Cloudflare Full (strict) TLS.
- OpenAI-compatible providers, SillyTavern character cards, memory privacy controls, and Lite mode.
- Windows and macOS desktop packages with ASAR integrity verification.

## Candidate Build Notice

These artifacts are not yet signed with commercial Windows or Apple developer certificates. Windows SmartScreen or macOS Gatekeeper may show an unknown publisher warning. Production signing and Apple notarization remain required before the stable release.
